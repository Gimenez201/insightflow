 InsightFlow

O InsightFlow é uma ferramenta web desenvolvida com foco em análise inteligente de dados, utilizando conceitos de engenharia reversa, inteligência artificial simulada e integração com Firebase.

---

 Objetivo

Transformar uma ferramenta existente em um produto autoral, adicionando funcionalidades inovadoras e utilizando tecnologias modernas como Firebase.

---

 Funcionalidades

-  Análise inteligente de dados (simulação de IA)
-  Histórico dinâmico de análises
-  Armazenamento em nuvem com Firebase Firestore
-  Exportação de relatórios em PDF
-  Interface moderna (Dark Mode + Glassmorfismo)

---

Diferenciais Competitivos

1. Integração com Firebase para armazenamento em tempo real  
2. Sistema de análise inteligente automatizada  
3. Exportação de relatórios em PDF  
4. Histórico persistente na nuvem  

---

Tecnologias Utilizadas

- HTML, CSS, JavaScript
- Firebase (Firestore)
- jsPDF
